# ForceMorphArea
An area that forces the player to enter a specific power-up state.
- `ForceMorphCube` (Center-origin cube shape)

## Arguments
|Name|Description|Default
|--- |---        |--- 
|Obj_arg0|Power-up state       |(Required)
|SW_A    |Enables/disables area|

## Power-up states
|Id |Name
|---|---
|0  |None
|1  |Rainbow Star
|2  |Fire Flower
|3  |Ice Flower
|4  |Bee Mushroom
|5  |Spring Mushroom
|6  |Boo Mushroom
|7  |Red Star
