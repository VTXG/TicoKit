# TimeAttackClock
A port of the Plus Clocks seen in SMG2.

## Arguments
|Name|Description|Default
|--- |---        |--- 
|Obj_arg0 |Time increase/decrease amount        |10
|Obj_arg1 |Remove time?                         |False
|Obj_arg2 |Enable shadow calculation?           |True
|Obj_arg3 |Color animation frame                |0
|SW_APPEAR|When enabled, makes the object appear|
