# RedBlueTurnBlock
A port of the Flip-Swap Panels seen in SMG2.

## Arguments
|Name|Description|Default
|--- |---        |--- 
|Obj_arg0|Rotation duration|40
|Obj_arg1|Target rotation X|0
|Obj_arg2|Target rotation Y|0
|Obj_arg3|Target rotation Z|180
|SW_A    |When enabled, makes the panel flip|(Required)
