# MarioConst
TicoKit unhardcodes the `MarioConst` tables, which are tables containing multiple player-related values and parameters.

The `MarioConst` binary files can be found inside `/PlayerData/`, under the names `MarioConst.bin` and `LuigiConst.bin`. To edit them, use [MarioConstTool](https://github.com/Lord-G-INC/MarioConstTool).

> [!NOTE]
> These files are not required for TicoKit to work. Not having them in your mod will simply result in the game using the default const values.
